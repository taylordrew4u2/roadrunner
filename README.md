# Roadrunner
Shared travel planner web app

## Quick Start
1. Run `npm install`
2. Run `npm run dev`
3. Open http://localhost:3000
4. Share trip invite links with group members

## Deploy to Vercel
1. Push code to GitHub
2. Connect repo to Vercel
3. Add environment variables in Vercel project settings (copy from `.env.local`)
4. Deploy

## Features
- ✅ Create trips with dates, locations, activities
- ✅ 24-hour timeline per day with event locations
- ✅ Google Maps with all pins + directions
- ✅ Upload tickets (images/PDFs)
- ✅ Shared notes & to-do lists (all members must check off tasks)
- ✅ Real-time sync across all users
- ✅ Invite members via shareable links
